## OBJETIVO
Desenvolver um sistema de controle de entrada para um evento

## DEVE SOLICITAR
Nome da Pessoa
Idade
Possui Convite? (Sim/Não)

## REGRAS
Idade < 16 -> Entrada negada
Idade > 16 + convite -> Entrada permitida
Sem convite -> Entrada negada

## PARA PENSAR
1. Uso do while para manter o sistema ativo;
2. Múltiplas condições (Decisão com if, elif e else)
